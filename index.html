git commit

git status

gitの状態を確認する

git diff

ローカルのステージとワークツリーの変更を確認する

git diff --staged

ステージとリポジトリの変更を確認する

git log

変更履歴を確認する

git log --online

変更履歴を1行で表示確認する

git log -p index.html

変更差分を表示する

git log -n コミット数

表示するコミット数を制限する

git rm ファイル名

ファイルを削除

git rm -r ディレクトリ名

ディレクトリを削除

git rm --cached ファイル名

ファイルを残したいとき
