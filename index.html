git commit

git status

gitの状態を確認する

ローカルのステージとワークツリーの変更を確認する

git diff

ステージとリポジトリの変更を確認する

git diff --staged

変更履歴を確認する

git log

変更履歴を1行で表示確認する

git log --online

変更差分を表示する

git log -p index.html

表示するコミット数を制限する

git log -n コミット数

ファイルを削除

git rm ファイル名

ディレクトリを削除

git rm -r ディレクトリ名

ファイルを残したいとき

git rm --cached ファイル名

消したファイルを元に戻す

git reset HEAD index.html

git checkout index.html
